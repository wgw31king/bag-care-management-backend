包包洗护门店管理系统 — Windows 一体化部署
========================================

本安装包已合并：
  backend\   后端程序（已编译 dist + 数据库脚本）
  frontend\  前端项目源码（安装时自动 npm build）
  deploy\    一键脚本

客户电脑只需安装：Node.js 20 + Docker Desktop


【首次部署】按顺序双击
----------------------------------------
1. deploy\00-一键安装.bat
   - 启动 PostgreSQL（Docker）
   - 安装后端依赖、建表、创建管理员 admin/admin
   - 安装前端依赖并编译页面

2. deploy\01-启动系统.bat
   - 浏览器打开 http://localhost:3001
   - 登录后请立即修改 admin 密码

【日常使用】
----------------------------------------
启动: deploy\01-启动系统.bat
停止: 关闭黑色窗口；长期不用可 deploy\02-停止数据库.bat
