包包洗护门店系统 - Windows 部署说明
========================================

【脚本闪退、看不到报错】
  请双击: deploy\install.bat 或 根目录 安装.bat
  窗口会保持不关；失败时查看 install-log.txt

【脚本】
  deploy\install.bat   首次安装
  deploy\start.bat     日常启动
  deploy\stop-db.bat   停止数据库
  安装.bat / 启动.bat  根目录快捷方式

需安装: Node.js 18+ 、 Docker Desktop（建议开机自启）

【首次】
  1. 解压完整 zip（须有 backend 和 frontend 文件夹）
  2. 双击 deploy\install.bat
  3. 等待 15-30 分钟
  4. 双击 deploy\start.bat
  5. 浏览器 http://localhost:3001  账号 admin 密码 admin
